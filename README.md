## Purpose

Minimal GitHub user lookup — a beginner-friendly version focused on search and profile display.

## Use Cases

- First API project in React
- Simple search UX
- Profile card layout practice

## Tech Stack

- React 16
- React Router
- Axios
- Bootstrap

## How to Run Locally

```bash
npm install && npm run dev
```

> On Node 17+, if build fails: `export NODE_OPTIONS=--openssl-legacy-provider`
